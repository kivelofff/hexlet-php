Вес Хэмминга Испытание курса: PHP: Массивы

Вес Хэмминга это количество единиц в двоичном представлении числа.

src/Solution.php

Реализуйте функцию hammingWeight, которая считает вес Хэмминга.

<?php

hammingWeight(0);
// → 0

hammingWeight(4);
// → 1

hammingWeight(101);
// → 4
