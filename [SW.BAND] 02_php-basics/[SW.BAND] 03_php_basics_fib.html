Фибоначчи Испытание курса: PHP: Основы

src/Solution.php

Реализуйте функцию fib находящую положительные числа Фибоначчи. Аргументом функции является порядковый номер числа.

Формула:

f(0) = 0
f(1) = 1
f(n) = f(n-1) + f(n-2)

Пример:

<?php

fib(3); // 2
fib(5); // 5
fib(10); // 55
